Job Specifications

  • Role: Junior Web Developer
  • Job Description:
    • Developing, testing, and maintaining front-end and back-end components of web applications.
    • Collaborating with designers, developers, and project managers to enhance user experiences.
    • Debugging and optimizing code for performance and scalability.
    • Staying up to date with emerging technologies and industry trends.
  • Skills Required:
    • Front-End: HTML, CSS, JavaScript, React
    • Back-End (Bonus): Node.js, Express, MongoDB
    • Version Control: Git, GitHub
    • Debugging & Testing: Chrome DevTools, Jest
  • Soft Skills:
    • Problem-solving: Ability to analyze issues and implement efficient solutions.
    • Teamwork: Work collaboratively with designers, developers, and project managers.
    • Communication: Clearly convey ideas and technical details to team members.
    • Adaptability: Willingness to learn new technologies and improve skills.
  • Experience & Education:
    • Bachelor's degree in Computer Science, Software Engineering, or related field (or equivalent experience).
    • Recent graduates or self-taught developers with strong portfolios are encouraged to apply.
  • Job Benefits:
    • Competitive salary with performance-based bonuses.
    • Remote work flexibility (hybrid option available).
    • Learning & development opportunities, including paid online courses.
    • Access to company hackathons and mentorship programs.
